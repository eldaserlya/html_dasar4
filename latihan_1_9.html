Pengalaman Pertamaku Belajar Membuat Website

Aku adalah mahasiswi prodi Teknologi Informasi yang sudah memasuki semester 2, aku berkuliah di Politeknik Negeri Madiun. Di semester 2 ini aku belajar membuat website pada mata kuliah Desain & Pemrogramman Web. Aku belajar membuat website dengan menggunakan IDE VS Code, juga github. Pada awal belajar membuat website yaitu membuat akun github terlebih dahulu kemudian menskinkronkan akun github dengan VS Code kita. belajarnya diawali dengan mempelajari elemen dasar HTML, mulai dari belajar struktur HTML, membuat warna pada background website, belajar Heading untuk menandai judul atau bagian penting dari sebuah HTML, dan belajar berbagai kode penulisan teks. Berikut adalah beberapa hasil dari proses belajarku tentang dasar-dasar HTML.
